# EmployeeDetails

## Introduction
On this project queries were made to extract necessary data from the employees database. Using SQL, information regarding employeesâ€™ gender, department, salaries, birth date , etc were given from various tables in the employees database using join function. Additionally, new columns were also made to generate calculated information such as age, duration as a manager, average employees growth, etc. The dashboard summarizes the difference between genders and also each department regarding the total number of employees, manager, age, and average wage.
